Select type

  • 콤보박스는 기본 select와 script를 사용한 select, 그리고 list-type으로 구현된 셀렉트 스타일 3가지를 제공한다.
  • Custom ComboBox 는 $('****').uChSelect() 로 호출해 줍니다.
  • list-type 의 콤보박스 사용시 웹 접근성을 고려하여 h3 > a[href] 의 hash(#...) 값과 ul 의 id값을 동일하게 맵핑시켜줍니다.
