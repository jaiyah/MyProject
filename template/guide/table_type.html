기본 가로형 리스트 테이블

  • 모든 테이블의 정렬은 class tac(text-align:center), tal(text-align:left),tar(text-align:right) 로 적용합니다.
  • default th 의 정렬값은 center, default td의 정렬값은 left 입니다.
  • 예) 전체 테이블의 정렬값이 center 일 경우 table에 클래스 tac 적용(tal,tar)
  • 예) 행의 전체 정렬값을 정렬하고자 할 경우에는 tr 에 클래스를 적용합니다.
  • 각 열의 정렬값이 다를 경우에는 각각의 td 에 클래스를 적용합니다.
