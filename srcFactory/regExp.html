(escape 문자) 역슬래시(\)는 메타문자다. (일반적으로 문자 그대로 사용되지 않고 특별한 의미를 지니는 문자를 일컫는다.) 다시 말해서 역슬래시는 문자들이 문자그대로 해석되게 하며(이스케이프), 특수한 문자 시퀀스를 시작하는 데 사용된다. ^\$ => 소스상 첫번째 문자가 시작하는 위치에서 $문자를 찾는다. \$$ => $인 문자를 문자열 끝에 위치하는 문자를 찾는다. \\ => \(역슬래시)인 문자를 찾는다. .(point) 기호 => 마침표 문자는 모든 문자를 가리킨다.(어떠한 문자든, 공백이건 간에, 즉 아무 문자 하나와 일치한다.) 마침표(.)는 어떠한 문자나 알파벳, 숫자, 심지어 문장의 끝을 나타내는 문장부호로 쓰인 마침표(.)자체와도 일치한다. 단, 줄바꿈(newline)문자를 제외한 모든 문자와 일치한다. ...... => 여섯개의 문자 덩어리로 이루어진 것을 찾는다. O.K. \. => . 만 찾는다 [](square brackets) => 그룹하나([])가 문자 하나를 의미한다. 즉, 대괄호 기호는 문자 집합을 정의한다. 대괄호 안에 있는 문자는 모두 집합의 구성원이 되며, 집합에 속한 문자 가운데 하나가 일치한다.(집합에 속한 문자가 모두 일치할 필요가 없다) 팁 : 얻고자 하는 결과와 패텬이 일치하는지 확인하는 일은 매우 쉽지만, 진짜 도전 과제는 얻고 싶지 않은 결과도 일치하는지 확인하는 일이다. [A-z] 패턴 사용시 주의하자. 아스키문자 A와 아스키문자 z사이에 있는 모든 문자와 일치하는데 Z와 a사이에 속하는 아스키문자 중에는 여는 대괄호인 ([)와 캐럿(^) 같은 문자도 포함되기 때문이다. 문자의 범위를 지정할 때는 두 값중 더 큰 값이 앞에 나오면 동작하지 않는다. 예) [3-1] 하이픈(-)은 대괄호([]) 안에서만 메타문자인 특수한 메타 문자로 취급된다. 집합 밖에서 하이픈(-)은 단순히 문자 그대로 하이픈(-)과 일치하기 때문에 하이픈 문자에 굳이 역슬래시(\)를 붙일 필요가 없다. [^] => 대괄호 안에서 ^(캐럿)을 사용하면 부정(not)의 의미로써 제외하려는 문자집합을 지정할 수 있다. 즉, 대괄호 안에서 지정한 목록을 제외하기 정리 => 메타문자인 여는 대괄호,닫는 대괄호 []는 문자집합을 정의하는데, 그 집합 구성원 중에 한 문자라도 일치해야 한다. 문자 집합은 문자를 일일이 열거하거나 하이픈 문자를 사용해 범위를 설정할 수 있으며 캐럿문자는 지정한 문자들을 제외한 어떤 것들을 일치시킨다. 공백 문자 찾기 \s 서브 패턴 () => capture Group 중괄호안에 |(파이프)를 사용하면 |(파이프)를 구분자로 하여 그룹을 나타낸다.
