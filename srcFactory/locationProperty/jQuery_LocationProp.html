	// === offset 그룹 ===
	offsetParent() ==> Dom에서 offsetParent 프로퍼티와 같음
	outerWidth(), outerHeight() ==> Dom에서 offsetWidth, offsetHeight 프로퍼티와 같음
	
	position().left, position().top ==> Dom에서 컨테이너인 부모(offsetParent)를 기준으로 엘리먼트가 위치한 x,y 값
	
	// === scroll 그룹 ===
	scrollLeft(), scrollTop() ==> Dom에서 scrollLeft, scrollTop 프로퍼티와 같음 (스크롤된 값)
			
	// === client 그룹 ===
	offset().left, offset().top ==> Dom에서 getBoundingClientRect() 메서드와 같음 
	Document를 기준으로 엘리먼트의 left, top, right, top, width, height 를 모두 알아 낼수 있음
	
	width(), height()  ==> style에서 작성된 너비와 높이
	innerWidth(), innerHeight() ==> 엘리먼트의 너비와 높이(단, 마진,보더제외, 패딩,스크롤바는 포함)
