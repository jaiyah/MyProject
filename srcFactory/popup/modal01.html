Authors are encouraged to use the article element instead of the section element when it would make sense to syndicate the contents of the elements.The section element is not a generic container element. When an element is needed for styling purpose or as a convenience for scripting, authors are encouraged to use the div element instead. A general rule is that the section element is appropriate only if the element’s contents would be listed explicitly in the document’s outline.With very few exceptions, section should not be used if there is no natural heading for it.
